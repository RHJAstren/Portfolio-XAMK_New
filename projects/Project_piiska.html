--- Piiska ---

The Project Management website

Piiska is the project management website which is used in the IT-Department of XAMK Kotka campus. The website is essential for the project studies which are present in all the bachelor studies of the IT-Department. I was a part of the development team for a very brief moment in which I was unfortunately too busy to be part of.


As for what I was able to do was focus on studying about full stack development. The way that I accomplished it, was by taking part in Helsinki University's MOOC (Massive Open Online Courses) course aptly named Full Stack Open .

The frontpage of Full stack open

Problems

The biggest problem that I faced throughout the project was that I took too much work for myself so I wasn't able to work on it as much as I had hoped. The course itself was quite challenging. To start, I had no idea of how the React framework worked. After working on the Astro.js project, I understood how things worked.


The course itself is quite well balanced. From the 15 parts, parts 0-7 give enough exercises and knowledge to work on Piiska. It might be a lot of work, but it allows for a lot time to be spent on the project. While there were no major problems with the exercises, it was still very useful to be part of the Helsinki University MOOC Discord, to see if there were other students with the same problems as I had.


From the parts, I was able to do until Part 2e and stopped just before exercise 2.16. I am interested in picking up from where I had left off, but that I will probably do on my free time. As for what I was able to contribute for the project itself, was the README.md file. The file contains all suggestions and guides for a new developer that can then be modified as the project continues and changes hands between students and developers.

Conclusion

It was a shame that I was a little too ambitious a year ago, but now I have a better understanding of managing my time and what i should do next time I try to tackle web development and my own portfolio development.
