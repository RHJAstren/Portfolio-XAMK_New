The theme for 2024 XAMK Game Jam Fall the theme was socket. A kind of an abstract type theme for which we decided on something more simple.

The Game: What I worked on

This was my first Unreal Engine 5 project so I mainly worked on the basics. Making a useable and scalable UI and creating the small play area and player character were my main points of focus. The most technical thing I dipped my feet into was using easing to turn on the light in the game scene.

Point of view of the player. Put objects from the tray into the right sockets.
The play area from outside of the players view. Very simple but effective.

The problems

Biggest problem was with using the Blueprints that are provided in the Unreal Engine 5. The blueprints are confusing to read and require a sense of confidence to use. I wasn't able to get the C++ programming to work so for next time I will set up my editor so that I'll be able to use it 100 %.

Conclusion

This time the project was in the right scope of the whole event so it was already a step in the right direction. The project hit a roadblock once I had to use C++. For what ever reason the editor could not recognize the C++ code and would throw errors. Overall, it was an improvement from our previous project and we will keep improving over the year.
